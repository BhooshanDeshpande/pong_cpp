# pong_cpp
A simple game of pong written in C++ 

![](https://github.com/BhooshanDeshpande/pong_cpp/blob/main/pong.gif)


Prerequisites : SDL library
```
sudo apt-get install libsdl2-dev
```

** To directly run ** 
```
cd build 
./pong_game
```

** To build the game ** 
```
rm -rf build 
mkdir build && cd build 
cmake ..
make 
./pong_game
```
