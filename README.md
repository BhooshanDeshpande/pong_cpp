# pong_cpp
A simple game of pong written in C++ 

![Sample GIF](https://github.com/BhooshanDeshpande/pong_cpp/blob/main/pong.gif)

**Prerequisites**
```
sudo apt-get install libsdl2-dev
```

**Run** 
```
cd build 
./pong_game
```

**Build** 
```
rm -rf build 
mkdir build && cd build 
cmake ..
make 
./pong_game
```
